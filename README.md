# 🎬 Movie Search App

A modern movie discovery platform built using **React, TypeScript, Vite, Express.js, and TMDB API**. The application allows users to search movies, explore genres, view detailed movie information, discover cast and crew members, and maintain a personalized watchlist through a clean and responsive interface.

---

## ✨ Features

### 🔍 Movie Search

* Search movies instantly using TMDB's vast movie database.
* Real-time search results.
* High-quality movie posters and ratings.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Shashank-4k/movie-search-app.git
cd movie-search-app
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
TMDB_API_KEY=YOUR_TMDB_API_KEY
```

### Run the Application

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
